# 🃏 Blackjack Game (Vinte e Um)



Um divertido e clássico jogo de **Blackjack** desenvolvido com tecnologias web padrão (Vanilla JavaScript, HTML5 e CSS3). O objetivo é simples: teste sua sorte contra a "Banca", puxe as cartas certas e tente chegar o mais perto possível de **21** sem estourar!

## 📌 Sobre o Projeto

Este projeto foi construído para colocar em prática conceitos de lógica de programação com JavaScript e manipulação do DOM. Ele conta com um visual de mesa de cassino, sistema de compra de cartas, regras de vitória/derrota contra o Dealer (banca) e até mesmo uma trilha sonora imersiva inspirada no jogo *Balatro*.

## 🚀 Funcionalidades

* **Lógica de Jogo Completa:** O jogador pode puxar novas cartas ("Nova Carta") ou manter sua mão ("Parar").
* **Inteligência da Banca:** Se o jogador decide parar, o jogo calcula automaticamente quem venceu comparando os valores com as cartas do Dealer.
* **Condições de Vitória/Derrota:** Sistema que detecta Blackjack (21 exato), estouro de limite (>21), vitória, derrota ou empate.
* **Trilha Sonora:** Reprodução automática de música de fundo ao iniciar a partida.
* **Design Responsivo e Temático:** Interface inspirada em mesas de cassino reais usando fontes do Google Fonts (`Anton`, `Roboto`).

## 🛠️ Tecnologias Utilizadas

* **HTML5**: Estruturação semântica da aplicação.
* **CSS3**: Estilização, uso de Flexbox para alinhamento e importação de fontes externas.
* **JavaScript (ES6)**: Lógica do jogo, arrays para armazenar cartas, manipulação do DOM e eventos.
